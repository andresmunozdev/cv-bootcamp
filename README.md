
# CV Interactivo - Dr. Sheldon Cooper 

Este proyecto es un Currículum Vitae interactivo desarrollado como parte del Bootcamp de Desarrollo Web Full Stack JS. 
Presenta la información profesional del personaje ficticio **Sheldon Cooper** (Young Sheldon y The Big Bang Theory).

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica.
* **CSS:** Estilos personalizados.
* **Bootstrap 5:** Diseño responsivo (Grid system, Navbar, Utilidades).
* **JavaScript & jQuery:** Interactividad y manipulación del DOM.

## Funcionalidades Interactivas
1. **Modo Materia Oscura:** Botón para alternar entre tema claro y oscuro.
2. **Validación de Formulario:** Verifica que los campos sean válidos antes de enviar dejando un mensaje de OK o NOK.
3. **Bazinga:** Botón con alerta clásica del personaje.

## Estructura de Archivos
* `index.html`: Estructura principal.
    * Carpeta assets-css
        * `style.css`: Archivo con estilos personales.
    * Carpeta assets-js
        * `script.js`: Lógica de la aplicación.
    * Carpeta assets-img
        * `foto_cv.webp`: Imágen de perfil.
